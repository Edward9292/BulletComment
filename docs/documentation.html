bulletcomment.js

docmumentation

        BulletComment is a library that allow user to let DOM object (image, text, link, etc) floating around
        on the webpage. Can be use to display the information, news , add, or even use this library to build 
        barrage on the webpage.

Getting Started

Scripts that need to be included in the webpage.

        <script src='bulletcomment.js'></script>

Create the BulletComment object

Set idOfDOM be a string representing the id of the DOMScope that wants to display the effect.

        let bc = new BulletComment(idOfDOM);

User can have an options to set the values of BulletComment

MAX_COL represent the maximum number of item can be exits in a same time in a row

MAX_ROW represent the maximum number of row can be exits in a same time

        options = {
            MAX_COL: 3,
            MAX_ROW: 2
        }
        let bc = new BulletComment(idOfDOM, options);

Start displaying comments.

        bc.start();

APIs

.start()

Start displaying the comments/items from queue

                let bc = new BulletComment(idOfDOM, options);
                bc.start();

.send(DOM_object)

Send the DOM_object into the queue and wait for displaying

                let comment = "Hello";
                bc.send(comment);

.keepSending(DOM_object)

Keep sending the DOM_object into the queue

                let comment = "Hello";
                bc.keepSending(comment);

Also, user have an option to set the time interval between two item been sended

The default value is 1000 (1 sec)

                let comment = "Hello";
                let timeInterval = 3000;
                bc.keepSending(comment, timeInterval);

.clear()

Cleaer the queue of the BulletComment object

                bc.clear();
